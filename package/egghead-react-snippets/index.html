Egghead React Snippets

About

This is a port of Joe Maddalone’s comprehensive snippets collection ReactSublimeSnippets, made with the help of atomizr.

If you’ve seen Joe’s videos on Egghead, you’ve probably seen him use these snippets in Sublime Text. I wanted to make these available for Atom users.

Snippets

Go to ReactSublimeSnippets for a table of all the snippets.
