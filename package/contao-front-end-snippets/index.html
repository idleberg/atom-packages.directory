Atom Contao Front-End Snippets

Atom snippet library for Contao Front-End Classes. Compatible with Contao 4.2.X and 3.5.X.

Installation

Packages

  1. Open “Install” in “Settings” and find and install “Contao Front-End Snippets” plugin.
  2. Restart Atom Editor (if required).

Git

  1. Clone the repository into your Atom package directory ~/.atom/packages.

git clone https://github.com/marcobiedermann/atom-contao-front-end-snippets.git

Download

  1. Download the .zip file and unzip it.
  2. Move the content into your Atom package directory ~/.atom/packages.

Usage

Start typing ce_, layout_ or mod_ in CSS, Less, Sass, Scss or Stylus files and the autocomplete windows opens.
