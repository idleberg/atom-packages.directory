Flexible by Chad Donohue

Install with:

Flexible UI

Flexible is an adaptable UI theme for Atom that is based off of Nucleus Dark UI

> Shown here with the Firewatch theme

Features

  • Improved file drawer using scope lines and file indicators
  • Use of the Lato typeface for readability
  • Improved contrast, padding, and margins

###### Searching ###### Create/Open File ###### Helper Messages

Installation

Go to settings within Atom, select Install and search for Flexible under themes.

Or install directly from the console:

apm install flexible-ui
