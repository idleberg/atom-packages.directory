Northem Dark - Atom Syntax

Description

North-bluish dark theme based on the same named color palette Northem Dark.
This is a darker and more blue-tinged variation of the Northem color palette.

Recommendations

This theme matches the Open Source font Source Code Pro.
Visit the GitHub Repository for more information.

The colored file icons on the preview screenshot are provided by the file-icons package.

Installation

Download and install the recommended font, read section Recommendations for more information.

Install via apm: apm install northem-dark-atom-syntax
Alternatively open Settings > Install, set the filter to Themes and search for northem-dark-atom-syntax.

Activate it by going to the Settings > Themes section and select Northem Dark from the Syntax Theme drop-down menu.
