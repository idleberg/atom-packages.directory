atom-isort

Atom.io plugin to format and sort Python imports using isort.

Prerequisites

Make sure you have isort installed and the correct path to the binary set in the package’s configuration.

Acknowledgements

Heavily influenced by Benjamin Hedrich’s atom-python-isort as well as blacktop’s atom-python-yapf. This package is a more up to date and actively developed version of atom-python-isort.
