Install with: 
    apm install slush-poppies

Slush & Poppies Atom Syntax Theme

A port of the popular Textmate and Sublime Theme. Found here: https://github.com/geoffgarside/TextMate-Themes/blob/master/Slush%20%26%20Poppies.tmTheme

Pair this up with the UI Theme https://atom.io/themes/unity-ui and you’ll feel like you’re back in Sublime or Textmate.

You can get a sense of what this syntax theme looks like by checking this out: http://colorsublime.com/theme/Slush_and_Poppies
