linter-swift-package-manager

This package will lint your .swift files in Atom through Swift Package Manager (SPM).

Uses bits and pieces from linter-swiftc (thank you @kepler0!)

Installation

$ apm install linter-swift-package-manager

Settings

You can configure linter-swift-package-manager by editing ~/.atom/config.cson (choose Open Your Config in Atom menu):

coffeescript "linter-swift-package-manager": # The path to Swift. executablePath: "/usr/bin/swift"
